# WME+ Brief

Single page HTML site.

## Stack
- Static HTML
- Tailwind CSS
- Vanilla JS
- lucide icons

## Behavior
- Fullscreen sections with scroll snapping
- Fixed navigation and chapter index
- Client side only, no backend or auth

## PDF
- html2canvas + jsPDF
- Exports core sections only

## Run
- Open `index.html` locally  
- Or serve with any static server
